\#Курсовой проект\+: Сервер обработки данных

\#Описание Серверная часть клиент-\/серверной системы для обработки векторов данных с аутентификацией по MD5.

\#Функциональность
\begin{DoxyItemize}
\item Аутентификация клиентов по логину/паролю с использованием MD5 и соли
\item Прием бинарных данных (векторов чисел double)
\item Вычисление суммы квадратов для каждого вектора
\item Защита от переполнения
\item Логирование всех событий в файл
\item Конфигурация через параметры командной строки
\end{DoxyItemize}
